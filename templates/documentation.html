Google ADK Documentation

The Google Agent Development Kit (ADK) is a powerful toolset for building AI agents. Below you'll find key information and resources to help you get started with the ADK.

Getting Started

Prerequisites
  • Python 3.9 or higher
  • pip (Python package installer)
  • Git (for cloning repositories)
Installation

The main ADK package can be installed using pip:

pip install google-adk llama-index

Some ADK samples require additional dependencies like llama-index, which are now installed automatically by this environment.

First Steps
  1. Set up the ADK environment (already done in this training environment)
  2. Explore the sample agents in the ADK Samples repository
  3. Run the samples to understand their functionality
  4. Start modifying existing samples or create your own agents

Key Concepts

Agents

Agents are AI-powered entities that can perform specific tasks. In the Google ADK, agents are built using Python and can leverage various Google services.

Functions

Functions are the building blocks of agents. They define what actions an agent can take or how it processes information.

Tools

Tools are pre-built functions that agents can use to perform common tasks or interact with external services.

Prompts

Prompts are used to guide the AI model's responses. Well-designed prompts are crucial for building effective agents.
